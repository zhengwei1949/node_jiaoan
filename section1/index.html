掌握模块化只需要掌握三大块：如何创建一个模块+如何把当前模块的属性或方法导出（暴露）出去+如何引第三方模块

  1. 如何创建模块：在Node.js中一个文件就是一个模块
  2. 如何把当前模块的属性或方法导出（暴露）
//导出
var sum = function(a,b){
    return parseInt(a) + parseInt(b);
}

exports.sum = sum;

//或者
module.exports.sum = sum;
  1. 如何引第三方模块

require(这里面的叫模块标识符)

  • 如果是绝对路径,并且带扩展名(.js,.json,.node) --> 去找对应的文件
  • 如果是绝对路径，不带扩展名 --> .js > .json > .node
  • 如果是相对路径,,并且带扩展名(.js,.json,.node) --> 去找对应的文件
  • 如果是相对路径，不带扩展名 --> .js > .json > .node
  • 如果是require('a')这种形式的 --> 内置模块，比如fs、path、http之类的

09-Node.js基础-ES6-let和const使用规则-13:30

let相关知识点

  • let不存在变量预解析
//存在预解析，代码不会报错
console.log(flag);
var flag = 123;

//不存在预解析，代码会报错
console.log(flag);
let flag = 456;
  • let声明的变量不允许重复声明(可以重复赋值)
//会报错
let flag = 123;
let flag = 456;
console.log(flag);

if(true){
    var flag = 123;
}
console.log(flag);

if(false){
    var flag = 123;
}
console.log(flag);

if(true){
    let flag = 123;
}
console.log(flag);

{
    var a = 100;
    let flag = 2;
}
console.log(a);//100
console.log(flag);//报错

for(var i=0;i<3;i++){
    console.log(i);
}
console.log(i);

//for循环括号中声明的变量只能在循环体中使用
for(let i=0;i<3;i++){
    console.log(i);
}
console.log(i);

//在块级作用域中，变量只能先声明再使用
if(true){
    console.log(flag);
    let flag = 123;
}

const知识点

  • const不能重复声明、也不能重复赋值，其他的和let一样
const n = 1;
n = 2;

const应用

  • 引入其他模块的时候，一般用const
  • 常量用const,比如Math.PI

什么是解构赋值

解构赋值允许你使用类似数组或对象字面量的语法将数组和对象的属性赋给各种变量。这种赋值语法极度简洁，同时还比传统的属性访问方法更为清晰。

//数组解构赋值
let [a,b,c] = [1,2,3];
console.log(a,b,c);

//数组解构赋值
let [a=123,b,c] = [,2,];
console.log(a,b,c);

//对象解构赋值
let {bar,foo} = {bar:"itcast",foo:"hi"};
console.log(foo,bar);

//对象解构赋值
let {foo,bar} = {bar:"itcast",foo:"hi"};
console.log(foo,bar);

//对象解构赋值
//abc是别名，如果有了别名，那么原来的名字就无效了
let {foo:abc,bar} = {bar:"itcast",foo:"hi"};
console.log(foo,bar);

//对象解构赋值
let {cos,sin,random} = Math;
console.log(typeof cos);
console.log(typeof sin);
console.log(typeof random);

//字符串解构赋值
let [a,b,c,d,e] = "hello";
console.log(a,b,c,d,e);

let {length} = "hi";
console.log(length);

## 12-Node.js基础-ES6-函数相关扩展 13:38
- 参数默认值
```javascript
//之前的写法
function foo(param){
    let p = param || 'hello';
    console.log(p);
}

foo();
foo('hi');

//ES6的写法
function foo(param = 'nihao'){
    console.log(param);
}

foo();
foo('hi');
  • 参数解构赋值 ```javascript function foo(uname='lisi',age=12){ console.log(uname,age); }

foo('zhangsan',13);


```javascript
function foo({uname='张三',age=13}={}){
    console.log(uname,age);
}

foo({uname:'zhangsan'});
  • rest参数(剩余参数)
function foo(a,...param){
    console.log(a,param);
}

foo(1,2,3,5,6);

function foo(a,b,...param){
    console.log(a,b,param);
}

foo(1,2,3,5,6);
  • ...扩展运算符
function foo(a,b,c,d,e){
    console.log(a+b+c+d);
}

foo(1,2,3,4,5,6)

function foo(a,b,c,d,e){
    console.log(a+b+c+d);
}
let arr = [1,2,3,4,5];
foo.apply(null,arr);

function foo(a,b,c,d,e){
    console.log(a+b+c+d);
}
let arr = [1,2,3,4,5];
foo(...arr);
